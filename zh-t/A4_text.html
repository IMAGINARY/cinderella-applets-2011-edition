Using the program shown on this page you can freely generate symmetric wallpaper ornaments. By using the sliders you can control the color palette and the behavior of a pen. Using your finger you can draw within the rectangular frame in the middle of the screen. You can choose between different types of symmetry by using the 17 buttons on the lower right of the screen.

Mathematical background: If there exist translational symmetries in two different directions there are (amazingly) only 17 different types of structuraly different symmetry groups that can arise - the so called wallpaper groups. Using the program you can experience all these different groups.

Suggestions: How do the same input strokes look in different wallpaper groups? What happens if you connect two different centers of rotational symmetry by a single stroke?

To further explore: The program iOrnament for iPhone and iPad offers many ways to paint ornaments and a lot of background information - including a virtual world exhibition of the most beautiful ornaments.

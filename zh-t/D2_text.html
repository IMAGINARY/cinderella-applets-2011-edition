Using Platonic solids one can even create amazing spacial structures with fractal properties. To this end one places at the vertices of a certain Platonic solid smaller copies of the same Platonic solid. Repeating this process one obtains amazing objects. If one starts with a tetrahedron one gets a well-known fractal object: the Sierpinski tetrahedron.

In the program one can rotate the object by simply dragging it.
